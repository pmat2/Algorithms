# Algorithms
Set of algorithm to stay in practice

## Coin counter
Solution counts optimal way of giving the change.  
Possible to adjust set of available coins.

## Fibonacci sequence
Solution counts n-th number of Fibonacci sequence with  
O(n) time complexity
O(n) space complexity
